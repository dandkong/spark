# 更新日志

## 0.4.0

- 增加旁路多模型回答：同一条用户消息可通过 `@` 让其他模型生成一次候选回答。
- 旁路回答不进入后续主线上下文，并可在同一轮消息下切换查看。
- 用户消息改为纯文本展示，保留原始换行。
- 优化消息操作区：恢复用户消息操作按钮，重新生成按钮不再区分主/旁路。
- 拆分 Chat 部分逻辑，提取回答切换器、消息工具栏与对话分组工具。

## 0.3.0

- 聊天切换助手时保持会话挂载，避免正在生成的回复被中断。
- 增加附件发送与消息附件预览，支持图片、文件的粘贴、拖拽和选择。
- 设置页模型获取改为从 models.dev 实时拉取，并保留模型能力信息。
- 增加模型能力配置，支持附件、图片、推理、工具等能力标记。
- 优化设置页交互：API Key 显示/隐藏、通用设置按显示/对话分组。
- 增加可调整宽度的助手侧边栏，并持久化折叠状态和宽度。
- 增加上下文消息数设置，默认不限，可按最近消息数裁剪请求上下文。
- 更新 Streamdown 样式扫描配置，修复消息渲染样式缺失问题。
